# SUPER FÚTBOL ⚽

Fútbol arcade *top-down* jugable en el **móvil**, hecho con HTML5 Canvas. Tu equipo (azul) contra la máquina (rojo): controlas al jugador más cercano al balón, regateas, robas y marcas. Es una **PWA instalable** 📲.

## 🌐 Juégalo / Descárgalo en vivo
# 👉 https://samuelcatalanz123.github.io/futbol/

## 📲 Instalar en el teléfono
- **Android (Chrome):** menú ⋮ → "Instalar aplicación" (o el botón *📲 Instalar app*).
- **iPhone (Safari):** Compartir ↑ → "Añadir a pantalla de inicio".

## 🎮 Cómo jugar
- **Un dedo** para mover a tu jugador (controlas al azul más cercano al balón).
- **⚽ CHUTAR** a portería · **👟 PASE** a un compañero.
- Roba el balón acercándote al rival. ¡Marca más goles en 2 minutos!

## 🧰 Tecnología
HTML5 Canvas + JavaScript puro · PWA (manifiesto + service worker) · GitHub Pages.
